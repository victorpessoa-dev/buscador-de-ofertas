# Buscador de Ofertas

O **Buscador de Ofertas** ajuda a encontrar produtos no Mercado Livre e na Shopee de forma simples, reunindo ofertas relevantes em uma única tela.

O usuário pode pesquisar pelo nome de um produto ou colar o link de uma oferta. A aplicação busca opções semelhantes, organiza os resultados e mostra as informações mais importantes para facilitar a comparação.

## Funcionalidades

- Busca por nome de produto.
- Busca a partir de link do Mercado Livre ou da Shopee.
- Listagem de ofertas semelhantes.
- Comparação de preço atual e preço antigo, quando disponível.
- Exibição de desconto informado pela loja.
- Exibição de avaliação, quantidade de vendas e frete grátis, quando essas informações estiverem disponíveis.
- Filtros por marketplace, faixa de preço, desconto, avaliação e frete grátis.
- Ordenação por melhor oferta, menor preço, maior preço, avaliação e desconto.
- Paginação dos resultados.
- Botão de compra para abrir a oferta na loja.
- Preservação dos resultados ao voltar para a página.

## Como Usar

1. Digite o nome do produto ou cole um link no campo de busca.
2. Ajuste os filtros, se desejar.
3. Clique em **Buscar Ofertas**.
4. Compare os produtos encontrados.
5. Clique em **Comprar Agora** para acessar a oferta.

## Informações Exibidas

Cada produto pode mostrar:

- Imagem.
- Nome.
- Marketplace.
- Preço atual.
- Preço antigo.
- Desconto.
- Avaliação.
- Quantidade de vendas.
- Frete grátis.
- Indicação de loja confiável.

## Objetivo

O objetivo do projeto é facilitar a busca por boas ofertas, reduzindo o tempo necessário para comparar produtos entre marketplaces diferentes.

Os resultados priorizam ofertas com preço competitivo, boa reputação, vendas relevantes e informações úteis para decisão de compra.
